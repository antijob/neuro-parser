# User Flows

## Telegram Subscriber Flow

1. User opens bot and selects categories.
2. User configures country and region filters.
3. User receives relevant incident notifications.

## Analyst Flow

1. Analyst reviews incidents in admin/API.
2. Analyst validates relevance and status transitions.
3. Analyst monitors source quality and classifier outcomes.

## Operator Flow

1. Operator monitors task pipeline health.
2. Operator inspects failures and retries.
3. Operator applies configuration or source updates.
